# contacts-backend

### Summary
Frontend service to handle a list of contacts.
Uses `react.js`.

### How to run
Note: This FE service expects its [BE counterpart](https://github.com/thalesmiguel/contacts-backend) to be running on :3000
#### On your own
1. `git clone`
2. `cd into directory`
3. `yarn start`
